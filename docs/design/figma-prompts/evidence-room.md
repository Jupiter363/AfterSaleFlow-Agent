# 证据书记官室 Figma 提示词

## 产品语境

卡通证据书房。用户和商家在服务端 PT2H 时限内分别提交多模态证据；证据书记官
校验格式、来源、一致性和风险，建立双方可见目录与权限隔离的原件库，但不宣布责任。

## 桌面帧（1440 × 1024）

- 顶部固定栏：服务端两小时倒计时、USER/MERCHANT 完成状态、当前卷宗版本、
  证据书记官状态、传票信箱。
- 左侧：用户提交桌；右侧：商家提交桌；中央：共享证据书架。
- 底部：书记官对话、多模态上传托盘、完成举证按钮。
- 共享证据目录显示证据引用、类型、提交方、时间、目录摘要与可见性；原件区明确
  标注授权范围，禁止把无权限原件泄露给对方。
- 证据卡展示五级建议：
  `VERIFIED / PLAUSIBLE / SUSPICIOUS / REJECTED / NEEDS_HUMAN_REVIEW`。
- 每张卡分离“原始事实”“解析文本”“AI 摘要”“校验建议”，并注明
  “不构成真实性保证”。

## 必备状态帧

- USER 视图、MERCHANT 视图。
- 双方未完成、一方完成、双方提前完成并封卷。
- PT2H 到期、一方缺席、晚到证据只读提示。
- 上传中、解析中、需补充、冲突证据、权限受限。

## 视觉基线

书架、线索卡、书签、印章和柔和纸张层次；暖白 + 天蓝 + 珊瑚 + 绿色 + 紫色。
禁止网盘式文件表格、传统后台上传页和严肃暗色法庭。
