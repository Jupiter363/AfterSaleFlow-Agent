# 安全说明

## 报告问题

安全问题不得在公开 Issue 中披露。请向项目维护者私下报告，并提供复现步骤、
影响范围和建议缓解措施。

## 安全边界

- `.env`、API Key、服务密钥和密码禁止提交或写入日志。
- USER 与 MERCHANT 的资源访问必须校验归属。
- CUSTOMER_SERVICE 不能批准执行；只有 PLATFORM_REVIEWER 可以批准。
- Agent 只能分析和生成草案，不能调用执行工具。
- Tool Executor 必须校验审批记录、风险等级、幂等键和操作者角色。
- 上传文件必须校验扩展名、MIME、大小与内容类型。
- 原始证据和脱敏证据物理分桶，访问需审计。
- 用户和商家内容一律视为不可信输入，Prompt 禁止遵循绕过流程的指令。
